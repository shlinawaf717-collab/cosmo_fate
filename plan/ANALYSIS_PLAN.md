## 0. 研究问题

**主问题**：在现有 SN + BAO + CMB 距离数据约束下，"宇宙热寂"结论对暗能量动态演化假设有多脆弱？

本研究不宣称发现新的宇宙终局。交付物是一组预先定义的脆弱性度量（§7）：终局后验概率对 (a) 先验选择、(b) 状态方程参数化、(c) 数据组合 的敏感度，外加分类器的空假设校准和数据约束视界。

**预期结论的两种形态（预先声明，避免事后叙事）**：

- 形态 A：终局概率主要由数据决定（跨先验/参数化变化 < 1 个数量级）→ 结论是"终局分类在当前数据下已相对稳健"。
- 形态 B：终局概率主要由假设决定（跨先验/参数化变化 ≥ 1 个数量级，或终局判定落在约束视界之外）→ 结论是"热寂/Rip 概率是假设的函数，不是数据的函数"。

两种形态都是可发表结果；本计划不预设倾向。

## 1. 模型

背景膨胀（不含扰动；结构增长数据不在本期范围内）：

```
H²(a)/H₀² = Ω_r a⁻⁴ + Ω_m a⁻³ + Ω_k a⁻² + Ω_DE · f_DE(a)
```

- `Ω_r` 固定：由 T_CMB = 2.7255 K、N_eff = 3.044 导出（光子 + 无质量中微子近似；中微子质量效应在距离数据下忽略，此近似作为已知局限写入论文）。
- `Ω_DE = 1 − Ω_m − Ω_k − Ω_r`（归一化闭合，非独立参数）。

**基线参数化（CPL）**：

```
w(a) = w₀ + wₐ(1 − a)
f_DE(a) = a^{−3(1 + w₀ + wₐ)} · e^{−3 wₐ (1 − a)}
```

嵌套检验点：ΛCDM = (w₀, wₐ) = (−1, 0)；wCDM = (w₀, wₐ=0)。

**参数化轴的对照模型**（§7 F_param 用，全部在此冻结）：

| 代号 | w(a) | 备注 |
|---|---|---|
| CPL | w₀ + wₐ(1−a) | 基线 |
| BA | w₀ + wₐ·z/(1+z²) （Barboza–Alcaniz） | 未来行为有界 |
| JBP | w₀ + wₐ·z/(1+z)² （Jassal–Bagla–Padmanabhan） | 未来趋于 w₀ |
| BIN4 | 分 4 bin 常数 w：z ∈ [0,0.3), [0.3,0.7), [0.7,1.5), [1.5,z_CMB]；未来区 a>1 延用第一 bin 值 | 约束视界的主要工具 |
| GP | 高斯过程重建 w(a)，核 = 平方指数，超参数 (σ_f, ℓ) 层级化采样，均值函数 = −1 | 次要工具；核选择敏感性单独报告 |

## 5. 终局分类判据（数学定义，冻结）

对每个后验样本，从 a=1 起向前积分背景方程至 a_max = 10⁴。

判定顺序与判据（按优先级）：

1. **CRUNCH（坍缩）**：∃ a ≥ 1 使 H²(a) ≤ 0。（仅曲率扩展或 ρ_DE 可负的参数化可触发；CPL 平坦基线下不可达，此分支为分类器完备性而设。）
2. **RIP（有限时间发散）**：t(a_max) − t₀ = ∫₁^{a_max} da/(a·H(a)) 收敛。数值判据：积分值在 a_max → 2·a_max 时相对变化 < 10⁻³。
3. **DS（de Sitter 渐近 / Λ-like）**：ρ_DE(a_max) > 0 且 |w(a_max) + 1| < 0.01。
4. **DECAY（暗能量衰退）**：ρ_DE(a_max)/ρ_DE(1) < 0.01，或 ρ_DE(a_max)/ρ_m(a_max) < 1。
5. 以上皆不满足 → **OTHER**（人工检查；OTHER 占比 > 1% 视为判据缺陷，触发修订流程）。

**边界标记**：任一样本若在阈值 ×2 或 ÷2 时分类翻转，标记 `boundary=True`；报告各终局概率时同时给出边界样本占比。

**热力学映射（冻结）**：

| 动力学类 | 热力学终局 |
|---|---|
| DS, DECAY | 热寂兼容（永恒膨胀） |
| RIP | 非热寂（有限时间奇点） |
| CRUNCH | 非热寂（坍缩） |

论文中明确声明：DECAY ≠ 逃脱热寂——这是本研究对现有终局文献分类学的修正点之一。

## 6. 统计规则（冻结）

- 采样器：MCMC 用 Cobaya（MH 或 NUTS 均可，Phase 1 定型后登记）；证据计算用 nested sampling（dynesty 或 PolyChord）。
- 收敛：Gelman–Rubin R−1 < 0.01（最差参数）；报告各链有效样本数。
- 显著性：嵌套模型用 Wilks Δχ²，**报告 p 值**，不使用"约 Nσ"的模糊表述；模型比较报告 ΔAIC、ΔBIC 与 ln(Bayes factor)（Jeffreys 尺度解读）。
- 终局概率 = 后验样本在该类的占比，**必须附带**：MC 采样误差（分批均值法）+ 边界样本占比。凡 P < 1% 的尾部概率，须用 nested sampling 复核后方可写入正文。

## 7. 脆弱性度量（核心交付物，预先定义）

对每个度量，目标统计量是 **P(RIP) 与 P(热寂兼容) 的变化幅度**（log₁₀ 跨度）：

1. **F_prior** = max − min of log₁₀ P(RIP)，跨先验集 {P1, P2, P3}，固定 CPL + D0。（注：P2/P3 下 P(RIP) 恒为 0 是合法结果，报告为"由先验排除"。）
2. **F_param** = 同上，跨参数化 {CPL, BA, JBP, BIN4, GP}，固定 P1 + D0。
3. **F_data** = 同上，跨数据组合 {D0…D4}，固定 CPL + P1。
4. **空假设校准**：以 D0 的 ΛCDM 最佳拟合为真值生成 **100 组** mock（从各似然的完整协方差抽样），跑完整管线（CPL + P1），报告各终局类的误判率 α。真实数据的终局概率必须与 α 对照解读。
5. **约束视界 a_h**：BIN4 参数化下，计算 w(a) 的后验-先验 KL 散度 D_KL(a)；a_h ≔ D_KL 首次降到 0.1 nat 以下的 a。报告"RIP/DECAY 判定所依赖的最早 a"与 a_h 的比值。GP 版本作为交叉检查，核超参数敏感性单独一节。

## 8. 阶段过关条件（Gate）

- **Gate 1（管线验证，硬性）**：复现 DESI DR2 + CMB + Pantheon+ 的已发表 w₀wₐCDM 结果——最佳拟合点落在已发表 68% 等高线内，ΛCDM 排除显著性与文献值（≈2.8σ）相差 ≤ 0.3σ。未过关不得进入任何真实数据的终局分析。
- **Gate 2（分类器校准）**：mock 校准中，ΛCDM 真值被判为"热寂兼容"的比例 ≥ 90%，且 OTHER 类占比 < 1%。
- **Gate 3（收敛）**：所有写入论文的链满足 §6 收敛标准。
